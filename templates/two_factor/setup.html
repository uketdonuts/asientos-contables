Configuración obligatoria de seguridad

La autenticación de dos factores es un requisito obligatorio para usar esta aplicación. No podrás acceder a las funciones del sistema hasta que la configures.

¿Qué es la autenticación de dos factores?

La autenticación de dos factores añade una capa adicional de seguridad a tu cuenta. Además de tu contraseña, necesitarás un código generado por una aplicación en tu dispositivo móvil.

Sigue estos pasos para configurar:
  1. Descarga una aplicación de autenticación como Google Authenticator o Microsoft Authenticator en tu dispositivo móvil.
  2. Escanea el código QR mostrado abajo o introduce la clave manualmente.
  3. Ingresa el código de 6 dígitos generado por la aplicación para verificar la configuración.
