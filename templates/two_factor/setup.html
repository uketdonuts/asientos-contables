Configuración Obligatoria

La autenticación de dos factores es un requisito obligatorio para usar esta aplicación. No podrás acceder a las funciones del sistema hasta que la configures correctamente.

¿Qué es la autenticación de dos factores?

La autenticación de dos factores (2FA) añade una capa adicional de seguridad a tu cuenta. Además de tu contraseña, necesitarás un código temporal generado por una aplicación en tu dispositivo móvil.

Sigue estos pasos para configurar:

  1. 1
    Descarga una aplicación autenticadora
    Instala Google Authenticator, Microsoft Authenticator o cualquier aplicación TOTP compatible en tu dispositivo móvil.
  2. 2
    Escanea el código QR o introduce la clave
    Usa la cámara de tu aplicación para escanear el código QR o introduce manualmente la clave secreta mostrada abajo.
  3. 3
    Ingresa el código de verificación
    Introduce el código de 6 dígitos que aparece en tu aplicación para verificar que la configuración es correcta.
